# Graph (DFS / BFS)

## 📌 Pattern Description

Graph problems involve nodes and connections (edges).
They can be represented using:

* Adjacency list
* Adjacency matrix
* Grid (treated as graph)

---

## 🧠 Core Concepts

* Depth First Search (DFS)
* Breadth First Search (BFS)
* Visited tracking
* Connected components
* Cycle detection

---

## 🎯 When to Use

Use graph traversal when you see:

* Grid / matrix problems
* Connected components
* Path finding
* Neighbor-based traversal

---

## 🔥 Key Problems

* #200 Number of Islands
* #133 Clone Graph

---

## 💡 Key Patterns

### 1️⃣ DFS (Recursive)

* Explore deeply before backtracking
* Used for connected components

### 2️⃣ BFS (Queue)

* Level-by-level traversal
* Used for shortest path problems

---

## ⚡ Tips

* Always track visited nodes
* Avoid infinite loops (especially in cycles)
* Use HashMap or Set when needed
