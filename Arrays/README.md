# Arrays & Hashing

## 📌 Pattern Description

Problems involving:

* Frequency counting
* HashMap / HashSet usage
* Basic array traversal

## 🧠 When to Use

* Duplicate detection
* Frequency-based problems
* Pair sum problems

## ⏱ Complexity Focus

* Aim for O(n) using hashing instead of O(n²)

## 🔥 Key Problems

* Two Sum
* Contains Duplicate
* Group Anagrams
